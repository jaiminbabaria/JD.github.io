⚙️ ONE-TIME FIREBASE SETUP

To play across different phones, this game needs a free Firebase database. Follow these steps once:

  1. Go to console.firebase.google.com
  2. Click Add Project → name it anything → continue
  3. In the left menu → Build → Realtime Database
  4. Click Create Database → choose any location → Start in test mode
  5. Copy the database URL (looks like https://xxx-default-rtdb.firebaseio.com)
  6. Paste it below and save
